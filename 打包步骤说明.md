# 检测数据统计工具 v2.2.1 打包步骤说明

## 环境准备
- Python 3.11+
- PyInstaller 6.20.0+
- 依赖包：pandas, openpyxl, pywebview, Pillow

## 完整打包流程

### 1. 安装依赖
```bash
pip install pandas openpyxl pywebview Pillow
```

### 2. 准备图标文件
将 `app_icon.ico` 放在项目根目录（与 Python 脚本同级）

### 3. 生成 spec 文件
```bash
pyi-makespec --onefile --noconsole --icon "app_icon.ico" --name "检测数据统计工具_v2.2.1" "检测数据统计工具_v2.2.1.py"
```

### 4. 编辑 spec 文件
修改生成的 `检测数据统计工具_v2.2.1.spec` 文件：

```python
# 在 Analysis 部分添加：
datas=[('app_icon.ico', '.')],
hiddenimports=['pywebview', 'pandas', 'openpyxl', 'xlrd', 'webview.platforms.winforms', 'clr'],
```

### 5. 执行打包
```bash
python -m PyInstaller "检测数据统计工具_v2.2.1.spec" --distpath "." --clean --noconfirm
```

或使用完整命令：
```bash
pyinstaller --onefile --noconsole --icon "app_icon.ico" --name "检测数据统计工具_v2.2.1" "检测数据统计工具_v2.2.1.py" --distpath "." --clean
```

## 关键步骤说明

### 图标嵌入验证
打包完成后验证图标是否成功嵌入：
1. 检查日志中是否有 `Copying icon to EXE` 行
2. 使用工具检查 EXE 文件图标资源

### 图标替换位置
1. **EXE 文件图标**：通过 PyInstaller `--icon` 参数设置
2. **窗口标题栏图标**：通过 `_set_window_icon()` 函数使用 Windows API 设置
3. **HTML 页面 favicon**：通过 base64 编码嵌入 HTML 模板

### 代码修改要点
1. 修改 `_generate_app_icon()` 函数直接返回图标路径
2. 修改 HTML 模板中的 favicon 链接为 base64 格式
3. 在代码中动态注入图标 base64 数据

## 常见问题解决

### 1. 图标不显示（Windows 缓存问题）
```powershell
# 重启资源管理器
taskkill /f /im explorer.exe
start explorer.exe

# 或清理图标缓存
del "%LOCALAPPDATA%\IconCache.db" /f /q
```

### 2. PyInstaller 打包失败
- 确保没有旧 EXE 文件占用：`taskkill /f /im "检测数据统计工具_v2.2.1.exe"`
- 清理缓存文件：删除 `build`、`dist`、`*.spec` 目录
- 以非管理员身份运行命令

### 3. 运行时依赖缺失
确保 spec 文件中包含所有隐式导入：
```python
hiddenimports=['pywebview', 'pandas', 'openpyxl', 'xlrd', 'webview.platforms.winforms', 'clr']
```
